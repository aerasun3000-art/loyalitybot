## 🚀 БЛИЖАЙШИЕ ЗАДАЧИ (Что делать сейчас)

### 1. Проверить статус системы
```bash
# Проверить, какие боты запущены
ps aux | grep python3

# Проверить логи
tail -f logs/client_bot.log
tail -f logs/bot_output.log
```

### 2. Запустить боты (если не запущены)
```bash
# Клиентский бот
nohup python3 client_handler.py > logs/client_bot.log 2>&1 &

# Партнёрский бот
nohup python3 bot.py > logs/bot_output.log 2>&1 &

# Админский бот
nohup python3 admin_bot.py > logs/admin_bot.log 2>&1 &
```

### 3. Проверить деплой
- ✅ Frontend на Netlify: https://loyalitybot.netlify.app/
- ✅ Backend на Fly.io: проверить статус через `fly status`

### 4. Использовать теги MoSCoW в Linear
```bash
# Создать теги MoSCoW
python3 linear_moscow_tags.py

# Проанализировать задачи
python3 linear_moscow_analyzer.py
```
